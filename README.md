# algo-partners
Fraud detection related data and scripts to share with partners.

## How to Run
1. Clone the repo
2. Install R packages:
   1. dplyr
   2. xlsx
   3. caret
3. Run `notebooks/predict.R`
4. Files:
   1. Input file is in `data/03_experiment_input/`
   2. Output files are in `data/04_experiment_output/`
